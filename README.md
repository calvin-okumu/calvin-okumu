### 🔧 What I've done

I'm Head of IT & Senior Systems Integration Specialist at **Comsec Security Systems**, where I design solutions and unify enterprise security infrastructure — access control, video management, and intrusion detection systems from CEM, Genetec, Milestone, Texecom, CCure, and others — while maintaining **99.5% uptime** across the environments I run.

That work is, at its core, systems integration: getting disparate platforms to expose clean interfaces and communicate reliably.

### 💻 What I'm building now

I'm pointing that same instinct at software — building backend services with:

- **Node.js** — self-hosted, self-managed deployments
